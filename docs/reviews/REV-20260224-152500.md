2. **Align LocalLlmLayer fallback** (Arch Finding 2): Change `LocalLlmLayer.generateSummary` empty-history fallback from `userMessages.join('. ')` to `generateFirstSentenceSummary(userMessages)` for consistency across all three layers.
3. **Add combined-rules ChatML test** (QA Finding 2): Test `'[INST]<|system|>be nice\x01[/INST]'` → `'be nice'` to lock in marker + control-character interaction.
4. **Add ADR-0016 callback wiring note** (Arch Finding 3): Add a Neutral consequence noting the mutable callback wiring window.
