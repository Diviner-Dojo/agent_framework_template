## Findings by Specialist

### QA Specialist
- **A-1 [Low]** — `isListening=true` helperText suppression branch has no regression test.
  Three of four helperText guards are now tested; `isListening=true` is the remaining gap.
  Recommendation: add one test with `VoiceLoopPhase.listening` using the same
  `voiceOrchestratorProvider.overrideWithValue` pattern as the `isSpeaking` test.
- **A-2 [Low]** — `find.byType(TabBar)` assertion does not verify chart rendered.
  `TabBar` lives above the chart in the widget tree; the real guard is `takeException()`.
  Documented limitation — acceptable given fl_chart's canvas rendering constraints.
- **A-3 [Low]** — `isWaiting` test uses `// ignore: invalid_use_of_protected_member`.
  Direct `StateNotifier.state =` mutation bypasses business logic and will fail to compile
  if `SessionNotifier` is migrated to Riverpod 2 `Notifier`. Add comment explaining
  why `overrideWith` is not used, or migrate to `overrideWith`.
- **A-4 [Low]** — Export retry loop runs max 15×500ms = 7.5s.
  On slow CI runners this may be tight. Increase to 20 iterations if flakiness appears.
- Confidence: 0.91

### UX Evaluator
- **UX-A1 [Low]** — `FileSystemException` SnackBar says "Check storage permissions" which
  is incorrect on Android API 29+ where WRITE_EXTERNAL_STORAGE is not required. Users will
  open the wrong settings panel. Rephrase to focus on available storage space.
- **UX-A2 [Low]** — Export button disabled state (`_isExporting=true`) is not announced as
  a live-region to screen readers. TalkBack users receive no auditory confirmation that export
  started. Add `Semantics(label: _isExporting ? 'Exporting data, please wait' : 'Export My Data')`.
- **UX-A3 [Low]** — Date labels in both chart builders clip at 200% text scale despite the
  `reservedSize: 36` increase. Wrap date `Text` in `MediaQuery(data: ..copyWith(textScaler:
  TextScaler.noScaling))` to cap scaling in the compact axis area.
- **UX-A4 [Low]** — Correlation tile `'move together/opposite'` phrasing is borderline
  plain-language. ADHD constraint is met; this is a future copy-pass opportunity.
- Confidence: 0.88

## Recommended Improvements (Non-Blocking)

1. **isListening helperText test** (QA A-1): Add `VoiceLoopPhase.listening` regression test
   to complete four-branch coverage of the `helperText` conjunction.
2. **FileSystemException copy** (UX-A1): Replace "storage permissions" with "storage space"
   to avoid misdirecting Android users to the wrong settings panel.
3. **Export button accessibility** (UX-A2): Add `Semantics` live-region wrapper on the
   export button for TalkBack users.
4. **Chart axis text scaling** (UX-A3): Cap `textScaler` to `TextScaler.noScaling` in both
   chart date label builders to prevent 200%-scale clipping.
5. **isWaiting test pattern** (QA A-3): Add comment or migrate to `overrideWith` for
   long-term refactor safety.
