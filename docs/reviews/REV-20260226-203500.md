## Required Changes Before Merge

1. **Fix completeError test assertion** (QA blocking): Change `test/utils/reusable_completer_test.dart` line 45-48 from `expect(() => completer.future, throwsA(isA<StateError>()))` to `await expectLater(completer.future, throwsA(isA<StateError>()))`. The current assertion tests a synchronous property access, not the awaited errored Future.

## Recommended Improvements (Non-Blocking)

1. **time_of_day server-side allowlist** (security, medium): Add allowlist validation in `buildChatSystemPrompt()` and `validateRequest()` for `context.time_of_day` (morning/afternoon/evening/late night). Also validate `days_since_last` and `session_count` as non-negative numbers.
2. **PTT delay test coverage** (QA, medium): Add widget tests using `FakeAsync` for the 800ms timer, double-tap cancel, and state-listener cleanup behavior.
3. **Extract voice state types** (architecture, medium): Move `VoiceSessionError`, `VoiceSessionErrorKind`, `VoiceLoopPhase`, and `VoiceOrchestratorState` to `lib/models/voice_session_state.dart`.
4. **Session summary mapping in DAO** (architecture, medium): Add `getRecentSessionSummaries()` method to SessionDao that returns `List<Map<String, String>>` with truncation, replacing inline mapping in session_providers.dart.
5. **Comment sessionSummaries ignore in LocalLlmLayer** (architecture, low): Add `// sessionSummaries ignored — see ADR-0023` to getGreeting() and getFollowUp().
6. **Test reset-cancels-timeout** (QA, medium): Add test verifying setTimeout before reset doesn't fire after reset.
7. **Document stripSessionDelimiters asymmetry** (security, low): Comment explaining why first pattern has no closing anchor.
8. **Add test for consecutive [PAUSE]** (QA, low): Document expected behavior for `[PAUSE][PAUSE]`.
