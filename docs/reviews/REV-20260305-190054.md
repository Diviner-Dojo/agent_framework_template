### UX Evaluator
- **F1 [Medium]** `helperText` 'Tap send icon to submit' persists when `isWaiting=true` or `isSpeaking=true` (field disabled — user cannot act on the instruction). Suppression should include `&& !isWaiting && !isSpeaking`.
- **F2 [Medium]** Vocabulary split: History tab uses integer labels (1/4/7/10), Trends tab now uses Low/Mid/High. Visually similar charts carry different axis semantics without contextual explanation. Recommended: apply semantic labels to the History chart too.
- **F3 [Low]** `reservedSize: 24` on rolling chart left axis may clip "High" at elevated text scale. Increase to 36.
- **F4 [Low]** Same root as F1: helperText briefly re-appears after listening stops but before `isWaiting` clears (flash of disabled-state affordance). Same fix applies.
- Confidence: 0.87
